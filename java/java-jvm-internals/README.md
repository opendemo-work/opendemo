# Java JVM Internals

Java虚拟机内部原理与实践。

## JVM架构

```
JVM架构:
┌─────────────────────────────────────────────────────────┐
│                    Java Source Code                      │
├─────────────────────────────────────────────────────────┤
│                    Compiler (javac)                      │
├─────────────────────────────────────────────────────────┤
│                    Bytecode (.class)                     │
├─────────────────────────────────────────────────────────┤
│  ┌─────────────────────────────────────────────────┐   │
│  │                    JVM                          │   │
│  │  ┌─────────┐  ┌─────────┐  ┌─────────┐       │   │
│  │  │ Class   │  │ Runtime │  │ Garbage │       │   │
│  │  │ Loader  │  │ Data    │  │ Collector│       │   │
│  │  └────┬────┘  └────┬────┘  └────┬────┘       │   │
│  │       └─────────────┴─────────────┘           │   │
│  │  ┌─────────────────────────────────────────┐   │   │
│  │  │          Execution Engine               │   │   │
│  │  │  ┌─────────┐  ┌─────────────────────┐  │   │   │
│  │  │  │ Interpreter│  │ JIT Compiler      │  │   │   │
│  │  │  └─────────┘  └─────────────────────┘  │   │   │
│  │  └─────────────────────────────────────────┘   │   │
│  └─────────────────────────────────────────────────┘   │
├─────────────────────────────────────────────────────────┤
│                    Native Methods                        │
└─────────────────────────────────────────────────────────┘
```

## 内存模型

```java
// 堆内存设置
java -Xms512m -Xmx2g -XX:+UseG1GC MyApp

// 查看GC日志
java -Xlog:gc*:file=gc.log MyApp
```

### 垃圾收集器选择

| 收集器 | 适用场景 | 特点 |
|--------|----------|------|
| Serial | 单核/小内存 | 简单高效 |
| Parallel | 高吞吐 | 多线程 |
| CMS | 低延迟 | 并发收集 |
| G1 | 大堆/平衡 | 分区收集 |
| ZGC | 超低延迟 | 亚毫秒级 |

## JVM调优

```bash
# 查看JVM参数
java -XX:+PrintFlagsFinal -version

# 常用调优参数
-XX:+UseG1GC                    # 使用G1收集器
-XX:MaxGCPauseMillis=200        # 最大GC停顿时间
-XX:+HeapDumpOnOutOfMemoryError # OOM时生成堆转储
-XX:HeapDumpPath=/path/to/dumps # 堆转储路径
```

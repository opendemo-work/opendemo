# Go结构化日志管理Zap实战Demo

## 简介

## 常见问题解答

**Q: zap 和 log 包有什么区别？**
A: zap 是结构化、高性能日志库，log 是标准库中的基础文本日志。zap 更适合生产环境和服务监控。

**Q: 如何将日志写入文件？**
A: 见 `advanced_logger.go`，通过配置 `WriteSyncer` 可将日志写入文件。

**Q: 为什么推荐使用 SugaredLogger？**
A: 它支持类似 `printf` 的格式化，开发更方便；但在性能敏感场景建议使用原始 `Logger`。

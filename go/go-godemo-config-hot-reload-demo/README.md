## 常见问题解答

**Q: 修改文件后没有反应？**
A: 确保编辑器保存的是同一个文件路径，某些编辑器会先写临时文件再移动，建议用命令行 `echo` 测试。

**Q: 如何监听整个目录？**
A: 将 `watcher.Add("config.json")` 改为 `watcher.Add("./configs/")` 即可。

**Q: 能否支持YAML/TOML格式？**
A: 可以！只需在重载时用对应解析库（如`go-yaml/yaml`）读取内容即可，监听机制不变。

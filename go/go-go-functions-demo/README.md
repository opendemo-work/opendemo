## 代码解析

### main.go
- 演示了函数的基本定义语法
- 展示多返回值用于错误/状态传递的Go惯用法

### closure.go

## 常见问题解答

**Q: 为什么Go函数可以返回多个值？**
A: 这是Go语言的设计特性，常用于返回结果的同时返回错误或状态信息。

**Q: 闭包中的变量生命周期是怎样的？**
A: 即使外层函数已返回，只要闭包还被引用，被捕获的变量就会继续存在。

**Q: 如何在Go中实现类似Python的装饰器？**
A: 可以使用高阶函数包装另一个函数，实现日志、认证等横切关注点。

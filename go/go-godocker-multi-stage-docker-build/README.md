## 常见问题解答

**Q: 为什么使用alpine镜像？**
A: Alpine是超小Linux发行版，基础镜像仅~5MB，能显著减小最终镜像体积。

**Q: CGO_ENABLED=0的作用？**
A: 禁用CGO使Go编译为完全静态二进制，不依赖系统glibc，可在任何Linux系统运行。

**Q: 构建时报权限错误？**
A: 确保Docker服务正在运行，并且当前用户有权限访问Docker daemon。

**Q: 如何减小构建时间？**
A: 利用Docker缓存，将`go mod download`放在单独层，仅在go.mod变更时重新下载依赖。

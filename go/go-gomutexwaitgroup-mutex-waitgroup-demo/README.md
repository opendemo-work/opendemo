## 常见问题解答

**Q: 为什么需要Mutex？**
A: 多个goroutine同时读写同一变量会导致数据竞争。Mutex确保同一时间只有一个goroutine能访问临界区。

**Q: WaitGroup的作用是什么？**
A: 它让主goroutine等待其他goroutine完成任务，避免程序提前退出。

**Q: 是否可以在defer中调用WaitGroup.Done()？**
A: 可以，且是推荐做法，确保即使发生panic也能正确释放。

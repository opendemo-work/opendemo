#### 1.2 Python 测试配置
**文件**: `configs/pyproject.toml`

**配置内容**:
- pytest 配置 (测试发现、标记、覆盖率)
- black 代码格式化配置
- isort 导入排序配置
- flake8 Lint 配置
- mypy 类型检查配置
- coverage 覆盖率配置

**使用方式**:
```bash
# 复制到项目根目录后
cp configs/pyproject.toml ./
pytest -v                    # 运行测试
pytest --cov=python          # 带覆盖率
black python/                # 格式化代码
flake8 python/               # 代码检查
```

#### 1.3 Node.js 测试配置
**文件**: `configs/.eslintrc.js`, `configs/.prettierrc`, `configs/jest.config.js`

**配置内容**:
- ESLint: Node.js 推荐规则 + 自定义规则
- Prettier: 代码格式化配置
- Jest: 测试框架配置，包含覆盖率阈值

**使用方式**:
```bash
# 复制配置文件后
npx eslint nodejs/           # 代码检查
npx prettier --write nodejs/ # 格式化
npx jest                     # 运行测试
npx jest --coverage          # 带覆盖率
```

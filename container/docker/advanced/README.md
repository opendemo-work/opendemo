## 🐛 常见问题

### Q1：服务启动失败？

**A**：检查容器运行时是否已安装并正常运行，查看日志定位错误。

### Q2：权限不足？

**A**：确认当前用户是否有权限操作容器运行时，必要时使用 sudo 或将用户加入相应用户组。

### Q3：命令找不到？

**A**：确认工具已正确安装，并将其加入系统 PATH。

## 🏗️ Docker 多阶段构建

多阶段构建可以显著减小最终镜像体积，只保留运行所需的产物：

```dockerfile
# 构建阶段
FROM golang:1.21 AS builder
WORKDIR /app
COPY . .
RUN go build -o myapp

# 运行阶段
FROM alpine:latest
WORKDIR /app
COPY --from=builder /app/myapp .
CMD ["./myapp"]
```

相比单阶段构建，最终镜像体积可减少 90% 以上。

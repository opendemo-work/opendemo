## 📖 核心概念

### 1. UEFI

UEFI 是 Secure Boot 安全启动 的基础，正确理解和配置它是保障安全的前提。

### 2. Secure Boot

Secure Boot 直接影响系统的安全性和可用性，需要根据组织策略进行规划。

### 3. PK/KEK/db

PK/KEK/db 提供了关键的技术能力，支持安全机制的有效运行。

### 4. 启动链

启动链 关系到合规性和审计要求，是企业安全治理的重要组成部分。

### 安全建议

- 不要在生产环境中使用默认密码或密钥。
- 定期更新依赖组件到最新稳定版本。
- 对敏感配置使用密钥管理工具（如 Kubernetes Secrets、Vault）。

- 在修改配置前备份现有环境。

## 🐛 常见问题

### Q1：服务启动失败？

**A**：检查 Docker 和 Docker Compose 是否正常运行，查看日志定位错误。

### Q2：连接数据库失败？

**A**：确认数据库用户名、密码和连接地址正确，检查端口映射和网络配置。

### Q3：配置不生效？

**A**：确认配置文件路径正确，重启服务后加载最新配置。

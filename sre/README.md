## 📚 技术栈概览

站点可靠性工程（SRE）通过软件工程方法解决运维问题。本技术栈提供从理论基础到工程实践的完整学习路径，包含 10 个实战案例，覆盖以下主题：

- SLO/SLI 定义与管理
- 错误预算与发布策略
- 事件管理与 On-Call
- 混沌工程与韧性验证
- 灰度发布与特性开关
- 容量规划与事后复盘
- Runbook 自动化

完成本技术栈学习后，你将能够：

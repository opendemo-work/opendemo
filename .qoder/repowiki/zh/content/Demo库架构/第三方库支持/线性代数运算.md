## 常见问题解答
**Q: 运行时报错 'ModuleNotFoundError: No module named 'numpy''？**  
A: 请确保已正确安装依赖：`pip install -r requirements.txt`

**Q: 为什么不用A**(-1)来求逆矩阵？**  
A: 应使用`np.linalg.inv(A)`，因为`**`表示元素幂而非矩阵幂。

**Q: 如何判断矩阵是否可逆？**  
A: 可先检查行列式是否为零：`np.linalg.det(A) != 0`

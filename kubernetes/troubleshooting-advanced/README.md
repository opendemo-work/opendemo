## 故障排查流程

```
1. 问题定义 → 明确故障现象
2. 信息收集 → 日志、事件、指标
3. 问题定位 → 缩小故障范围
4. 根因分析 → 找到根本原因
5. 解决验证 → 应用修复并验证
```

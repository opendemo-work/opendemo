# Domain-8: Kubernetes可观测性

> **案例数量**: 27 个 | **最后更新**: 2026-03 | **专业级别**: 企业级生产环境 ⭐⭐⭐⭐⭐

## 概述

Kubernetes可观测性域全面覆盖监控指标、日志收集、分布式追踪、告警管理、SLO/SLI体系等核心技术。为企业构建生产级可观测性架构提供完整指导。

**核心价值**：
- 📊 **监控体系**：Prometheus、Grafana、指标采集
- 📝 **日志管理**：ELK/EFK、Loki、日志分析
- 🔍 **分布式追踪**：Jaeger、OpenTelemetry、链路追踪
- 🚨 **告警管理**：Alertmanager、智能告警、事件响应

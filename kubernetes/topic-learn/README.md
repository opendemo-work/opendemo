## 培训体系

### 内部培训 (inner-training)

**目标人群**: 阿里云 ACK 运维工程师
**培训周期**: 4 周
**培训形式**: 线下集中培训 + 实战项目

#### 课程大纲

| 周次 | 主题 | 内容 |
|------|------|------|
| 第1周 | ACK/ACR 生命周期 | 集群创建、升级、删除、证书管理 |
| 第2周 | 安全与监控 | RBAC、RAM集成、漏洞扫描、监控告警 |
| 第3周 | 节点与工作负载 | 节点管理、节点池、Pod管理、组件运维 |
| 第4周 | 网络与存储 | Service、Ingress、Terway、Flannel、存储管理 |

#### 实战项目

| 项目 | 描述 | 难度 |
|------|------|------|
| P1 | ACK 集群生命周期管理 | 中级 |
| P2 | 安全监控体系搭建 | 中级 |
| P3 | 节点与工作负载管理 | 中级 |
| P4 | 网络存储实践 | 高级 |
| P5 | 毕业项目 | 高级 |

### 公开培训 (public-training)

**目标人群**: Kubernetes 初学者和运维工程师
**培训周期**: 4 周
**培训形式**: 线上课程 + 实战练习

#### 课程大纲

| 周次 | 主题 | 内容 |
|------|------|------|
| 第1周 | 基础入门 | Docker、Linux、K8s架构、集群搭建 |
| 第2周 | 核心技术 | 控制平面、工作负载、网络、存储 |
| 第3周 | 运维实践 | 安全、可观测性、故障排查、平台运维 |
| 第4周 | 企业级实践 | 企业监控、GitOps、安全合规、最佳实践 |

#### 实战项目

| 项目 | 描述 | 难度 |
|------|------|------|
| P1 | K8s 集群搭建 | 初级 |
| P2 | 生产应用编排 | 中级 |
| P3 | 可观测性与故障演练 | 中级 |
| P4 | GitOps 流水线 | 高级 |
| P5 | 毕业项目 | 高级 |

## 部署前检查清单

### 硬件要求

```yaml
# 最小配置
control-plane:
  cpu: 2
  memory: 4Gi
  storage: 50Gi

worker:
  cpu: 2
  memory: 8Gi
  storage: 100Gi

# 生产配置
control-plane:
  cpu: 4
  memory: 16Gi
  storage: 100Gi

worker:
  cpu: 8
  memory: 32Gi
  storage: 500Gi
```

### 网络要求

- 节点间网络互通
- 开放必要端口（6443, 2379-2380, 10250-10252 等）
- 配置 DNS 解析

### 系统要求

- 操作系统：Ubuntu 20.04+, CentOS 7+, RHEL 8+
- 内核版本：4.19+
- 关闭 Swap
- 配置 sysctl 参数

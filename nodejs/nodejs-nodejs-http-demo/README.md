## 常见问题解答

**Q: 运行时报错 'Address already in use'？**
A: 说明端口被占用。请检查是否有其他程序占用了3000或3001端口，或修改代码中的端口号。

**Q: 客户端收不到数据？**
A: 确保`api-server.js`已先启动。客户端请求的是本地3001端口，服务未启动则连接失败。

**Q: 如何支持POST请求？**
A: 可以监听`'data'`事件收集请求体，然后在`'end'`事件中处理。本示例聚焦GET，但原理类似。

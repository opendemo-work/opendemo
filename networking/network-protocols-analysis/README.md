# Network Protocols Analysis

网络协议深度分析演示。

## OSI模型

```
OSI七层模型:
┌─────────────────────────────────────────┐
│ 7 应用层: HTTP, FTP, DNS, SMTP          │
├─────────────────────────────────────────┤
│ 6 表示层: TLS, SSL, 加密编码             │
├─────────────────────────────────────────┤
│ 5 会话层: NetBIOS, RPC                  │
├─────────────────────────────────────────┤
│ 4 传输层: TCP, UDP, SCTP                │
├─────────────────────────────────────────┤
│ 3 网络层: IP, ICMP, OSPF                │
├─────────────────────────────────────────┤
│ 2 链路层: Ethernet, ARP, PPP            │
├─────────────────────────────────────────┤
│ 1 物理层: 网线, 光纤, 无线               │
└─────────────────────────────────────────┘
```

## 协议分析

```bash
# TCP状态机分析
tcpdump -i eth0 'tcp[tcpflags] & tcp-syn != 0'

# DNS查询分析
dig +trace example.com

# HTTP协议分析
curl -v https://example.com
```

## 学习要点

1. 协议封装与解封装
2. TCP/UDP对比分析
3. 路由协议原理
4. 应用层协议解析

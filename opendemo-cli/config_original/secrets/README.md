# 🔐 密钥管理配置

## 目录结构
```
config/secrets/
├── local/          # 本地开发密钥
├── staging/        # 测试环境密钥  
├── production/     # 生产环境密钥
└── templates/      # 密钥模板
```

## 密钥轮换策略

### 自动化轮换
- JWT 密钥：每 90 天轮换
- API 密钥：每 180 天轮换
- 数据库密码：每年轮换

### 手动轮换流程
1. 生成新密钥
2. 更新配置
3. 重启服务
4. 验证功能
5. 废弃旧密钥

## 安全最佳实践

- ❌ 永远不要在代码中硬编码密钥
- ✅ 使用环境变量或密钥管理服务
- ✅ 定期轮换密钥
- ✅ 限制密钥访问权限
- ✅ 监控密钥使用情况

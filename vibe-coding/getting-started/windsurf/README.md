# Windsurf 快速上手

## 什么是 Windsurf

Codeium 推出的 AI-first IDE（基于 VS Code），特色是 Cascade 功能——多步推理 + 自动执行，适合复杂编码任务。

## 安装

1. 下载：https://codeium.com/windsurf
2. 登录 Codeium 账号

## 核心 Vibe Coding 功能

### Cascade（级联对话）
Windsurf 的核心差异化功能——AI 自动分析任务、规划步骤、逐步执行，过程中可以自动运行终端命令、读取输出、自我修正。

### Write 模式
AI 可以直接编辑文件，不需要你手动粘贴代码。

### 上下文感知
自动理解整个项目结构，不需要手动 `@` 引用文件。

## Vibe Coding 最佳实践

1. **用 Cascade 处理复杂任务** — 比如"给这个项目添加完整的测试覆盖"，让 Cascade 自己规划
2. **信任但要验证** — Cascade 会自动执行命令，但每步结果要检查
3. **先小后大** — 先用简单任务测试 Cascade 的理解，确认它理解项目后再给大任务

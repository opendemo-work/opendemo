## 使用方式

**跟练**：读 CHALLENGE.md → 用 AI 工具完成 → 对比 SOLUTION.md
**挑战**：只看目标约束 → 独立完成 → 对比 prompt 策略差异
**造题**：从 OpenDemo 其他 508 个 demo 提取新 CHALLENGE → PR 贡献

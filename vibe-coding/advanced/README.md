# 高级主题

完成基础练习后，这些是真实项目中会遇到的高阶场景。

## 四个方向

| 方向 | 解决什么问题 | 前置要求 |
|------|-------------|----------|
| [MCP 集成](mcp-integration/) | 让 AI 连接数据库、文件系统、GitHub 等外部工具 | 了解 MCP 协议 |
| [多文件重构](multi-file-refactoring/) | 一次 prompt 修改 5+ 文件，保持逻辑一致 | 熟练掌握迭代技巧 |
| [跨栈项目](cross-stack-project/) | 前端 + 后端 + 部署一体化 | 至少完成 3 个不同栈的 practice |
| [代码迁移](legacy-migration/) | 用 AI 把老代码迁移到新技术栈 | 了解源和目标两门语言 |

## 推荐顺序

1. **多文件重构** — 最实用的日常技能
2. **跨栈项目** — 综合能力提升
3. **MCP 集成** — 工具链扩展
4. **代码迁移** — 特定场景需要时再学

# Vibe Coding 基础技法

在开始做练习之前，先花 30 分钟过一遍这 5 个技法。它们是所有后续练习的基础。

## 学习顺序

```
Prompt Engineering → 迭代技巧 → AI Debug → AI 写测试 → AI Code Review
     (必修)          (必修)     (必修)     (选修)       (选修)
```

### 必修（前 3 个）

| 技法 | 学什么 | 时间 |
|------|--------|------|
| [Prompt Engineering](prompt-engineering/) | 5 种高效 Prompt 模式，避免反面模式 | 10min |
| [迭代技巧](iterative-refinement/) | 骨架→填充、测试驱动等 4 种迭代策略 | 10min |
| [AI Debug](debugging-with-ai/) | 5 种 Debug 模式，报错堆栈分析 | 10min |

### 选修（按需）

| 技法 | 学什么 | 时间 |
|------|--------|------|
| [AI 写测试](testing-with-ai/) | 4 种测试生成方法 + 不同语言模板 | 10min |
| [AI Code Review](code-review-with-ai/) | 3 种 Review 模式 + 重点检查清单 | 10min |

## 下一步

学完技法后，选一个 [practice](../practices/) 开始练习。

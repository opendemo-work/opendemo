# 🔥 BigData 大数据技术栈

> 大数据处理与分析实践，涵盖 Spark、Flink、ClickHouse、Kafka、数据仓库、数据湖和 ETL/调度等方向。

---

## 📚 技术栈概览

大数据技术帮助企业从海量数据中提取价值。本技术栈提供从批处理到实时流处理的完整学习路径，包含 7 个实战案例，覆盖以下主题：

- Spark SQL 批处理分析
- Flink 实时 ETL
- ClickHouse OLAP 分析
- Kafka 流处理
- 数据仓库分层设计
- Iceberg 数据湖
- Airflow ETL 调度

---

## 🎯 学习目标

完成本技术栈学习后，你将能够：

- ✅ 使用 Spark/Flink 处理批量和实时数据
- ✅ 使用 ClickHouse 进行 OLAP 分析
- ✅ 设计数据仓库分层架构
- ✅ 使用 Airflow 编排 ETL 流程
- ✅ 理解数据湖和现代数据栈

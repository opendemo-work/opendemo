# Virtualization Concepts Demo

虚拟化概念与原理深度解析。

## 什么是虚拟化

虚拟化技术允许在单一物理硬件上运行多个操作系统实例，实现资源的抽象、隔离和共享。

## 虚拟化类型

### Type 1 - 裸机虚拟化 (Bare-metal)

```
┌─────────────────────────────────┐
│  VM1    VM2    VM3              │
│ ┌───┐  ┌───┐  ┌───┐            │
│ │OS1│  │OS2│  │OS3│            │
│ └───┘  └───┘  └───┘            │
├─────────────────────────────────┤
│      Hypervisor (Type 1)        │
│    VMware ESXi, Xen, Hyper-V    │
├─────────────────────────────────┤
│         Physical Hardware       │
└─────────────────────────────────┘
```

**特点**:
- 直接运行在硬件上
- 性能高
- 用于数据中心

### Type 2 - 托管虚拟化 (Hosted)

```
┌─────────────────────────────────┐
│  VM1    VM2                     │
│ ┌───┐  ┌───┐                   │
│ │OS1│  │OS2│                   │
│ └───┘  └───┘                   │
├─────────────────────────────────┤
│  Hypervisor (Type 2)            │
│  VMware Workstation, VirtualBox │
├─────────────────────────────────┤
│         Host OS                 │
├─────────────────────────────────┤
│      Physical Hardware          │
└─────────────────────────────────┘
```

**特点**:
- 运行在Host OS之上
- 易用性好
- 用于开发和测试

## 虚拟化技术对比

| 技术 | 类型 | 代表产品 | 适用场景 |
|------|------|----------|----------|
| 全虚拟化 | Type 1 | VMware ESXi | 企业数据中心 |
| 半虚拟化 | Type 1 | Xen | 云计算平台 |
| 硬件辅助 | Type 1/2 | KVM | Linux环境 |
| 操作系统级 | - | LXC, Docker | 应用容器化 |

## 虚拟化 vs 容器

```
虚拟机 (VM)                    容器 (Container)
┌─────────────────┐           ┌─────────────────┐
│   Application   │           │   Application   │
├─────────────────┤           ├─────────────────┤
│   Bin/Libs      │           │   Bin/Libs      │
├─────────────────┤           ├─────────────────┤
│   Guest OS      │           │   Container     │
├─────────────────┤           │   Engine        │
│   Hypervisor    │           ├─────────────────┤
├─────────────────┤           │   Host OS       │
│   Host OS       │           ├─────────────────┤
├─────────────────┤           │   Hardware      │
│   Hardware      │           └─────────────────┘
└─────────────────┘
```

| 特性 | 虚拟机 | 容器 |
|------|--------|------|
| 启动时间 | 分钟级 | 秒级 |
| 资源占用 | GB级 | MB级 |
| 隔离级别 | 强 | 较弱 |
| 性能 | 接近原生 | 原生 |
| 密度 | 10-100 | 100-1000 |
